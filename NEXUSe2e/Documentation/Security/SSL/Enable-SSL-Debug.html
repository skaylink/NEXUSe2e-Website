Enable SSL Debug

NEXUSe2e SSL Ciphers

NEXUSe2e ciphers are limited by what Java provides it with in the cryptography libraries, as listed on their manual page.

Enable Java SSL Debug

To enable SSL debug for Java you need to add Djavax.net.debug=all to your NEXUSe2e service

  • Your service executable is located in your /bin folder in the Tomcat directory. It probably called something like NEXUSe2ew.exe.
     
  • Open the service, switch to the Java tab and add Djavax.net.debug=all to the Java options field. Apply or press OK.
Zoom
  • Start/restart your service

You should now be able to see SSL debug messages in your NEXUSe2e log.

For further information about SSL debugging please check the Java SE Security Documentation according to your installed Java version.
