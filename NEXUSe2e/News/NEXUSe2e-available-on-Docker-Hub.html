NEXUSe2e available on Docker Hub

  • Dec 12, 2016

Since our daily work is more and more influenced by agile services and high availibility environments we decided to make our NEXUSe2e docker container public.

It is available as a free public version. The base image is recommended for testing purposes only. However the container can be configured to be production ready assuming you provide the necessary database and system configuration.

A mySQL or postgreSQL container works fine, but your current dedicated database will also work. There are no code differences between the war file available at NEXUSe2e.org and the docker container available on docker hub.

We will provide additional services regarding configuration and maintenance of docker containers in test and production.
